# Melone

> Memory System by Clone

A local-first memory engine for your desktop. Melone records what happens on your desktop — screen, keyboard, mouse — turns it into searchable episodes, and makes that memory available to your AI agents over MCP.

> 🚧 **Pre-release.** Melone is in private development right now. The open-source release is coming soon — ⭐ star this repo to get notified.
